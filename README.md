# SQL Subqueries Practice

This repository contains an SQL file focused on **subquery concepts** used in relational databases.  
The queries are written for learning and practice purposes, especially useful for **Data Analyst / Data Science beginners**.

## 📂 File Included
- `sub quree.sql` – Contains SQL queries demonstrating the use of subqueries.

## 🧠 Topics Covered
- Subqueries in `SELECT` statements  
- Subqueries in `WHERE` clause  
- Nested queries  
- Using subqueries with comparison operators  
- Practical examples for data analysis scenarios  

## 🛠️ Tools & Technologies
- SQL
- Compatible with:
  - MySQL  
  - PostgreSQL (minor syntax adjustments may be needed)

## ▶️ How to Use
1. Open your SQL editor (MySQL Workbench, pgAdmin, DBeaver, etc.)
2. Create or select a database
3. Open `sub quree.sql`
4. Run the queries one by one to understand the output

## 🎯 Learning Objective
The goal of this file is to strengthen understanding of **subqueries**, which are commonly used in:
- Interviews
- Business data analysis
- Reporting and insights generation

## 🚀 Who Should Use This?
- Beginners learning SQL
- Data Analyst / Data Science students
- Anyone preparing for SQL interviews

## 📌 Note
This file does not contain any real or sensitive data.  
It is created purely for educational and practice purposes.
